# Automated Inventory Alert System

> Real-time inventory monitoring with low-stock alerts, a multi-page React dashboard, and automated CSV/JSON report generation.

---

## Tech Stack

| Layer     | Technology                          |
|-----------|-------------------------------------|
| Backend   | Python 3.13 · FastAPI · Pydantic v2 |
| Frontend  | React 18 · React Router v6 · Axios  |
| Database  | MySQL 8 (via SQLAlchemy + PyMySQL)   |
| Scheduler | Python `schedule` library            |

---

## Features

- **Dashboard** — summary cards (total products, low stock, out of stock, inventory value in ₹) and a colour-coded stock status table.
- **Inventory Management** — full CRUD with search-by-name/SKU, status filter, inline quantity editing, and add-product form.
- **Reports** — on-demand CSV or JSON report generation with a download history table (persisted in localStorage).
- **Alerts Panel** — slide-out notification panel (bell icon with badge count) showing out-of-stock and low-stock items, auto-refreshes every 30 s.
- **Background Scheduler** — standalone script that periodically checks stock levels, prints console alerts, and auto-generates CSV reports.
- **Pydantic Validation** — request/response schemas validated with Pydantic v2.
- **Graceful DB Handling** — the API server starts even when MySQL is unavailable; table creation is retried on the next restart.

## Getting Started

### Prerequisites

- Python 3.10+
- Node.js 18+
- MySQL 8 (can be configured later — the app starts without it)

### 1. Backend

```bash
cd backend
python3 -m venv venv
source venv/bin/activate          # macOS / Linux
pip install -r requirements.txt
```

Create a **`.env`** file (or edit the existing one):

```
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=inventory_db
LOW_STOCK_THRESHOLD=10
CHECK_INTERVAL_MINUTES=30
```

**Start the API server:**

```bash
uvicorn app.main:app --reload --port 8000
```

**Start the background scheduler** (separate terminal):

```bash
source venv/bin/activate
python scheduler.py
```

### 2. Frontend

```bash
cd frontend
npm install
npm start                         # opens http://localhost:3000
```

### 3. MySQL (when ready)

Create the database — tables are auto-generated by SQLAlchemy on first successful connection:

```sql
CREATE DATABASE IF NOT EXISTS inventory_db;
```

## API Endpoints

All endpoints are prefixed with **`/api`**.

| Method  | Path                        | Description                  |
|---------|-----------------------------|------------------------------|
| GET     | `/products`                 | List all products            |
| GET     | `/products/{id}`            | Get single product           |
| POST    | `/products`                 | Create product               |
| PUT     | `/products/{id}`            | Update product fields        |
| PATCH   | `/products/{id}/stock`      | Update stock quantity only   |
| DELETE  | `/products/{id}`            | Delete product               |
| GET     | `/alerts`                   | Low-stock alerts             |
| POST    | `/reports?fmt=csv\|json`    | Generate stock report file   |

Interactive API docs → **http://localhost:8000/docs** (Swagger UI)

## Database Schema

**`products`** table:

| Column               | Type         | Notes                       |
|----------------------|--------------|-----------------------------|
| id                   | INT (PK)     | Auto-increment              |
| name                 | VARCHAR(255) | Not null                    |
| sku                  | VARCHAR(100) | Unique, not null            |
| quantity             | INT          | Default 0                   |
| price                | FLOAT        | Default 0.0                 |
| low_stock_threshold  | INT          | Default from env config     |
| created_at           | DATETIME     | Auto-set                    |
| updated_at           | DATETIME     | Auto-updated                |
